# Architecture

Here's the whole system in one breath: Pulse is a **control-plane** — everything that happens _around_ a stream (accounts, follows, chat, presence, analytics) is event-driven, with Kafka as the backbone. The actual video and audio? Those bytes take a different road entirely — the **media plane** — and never touch Kafka. That's why you won't see them in the diagram below.

## System topology

Three services, deliberately written in three different languages so the contracts between them have to be _real_, sit on top of four infrastructure pieces. The diagram is honest about **build state**: the infrastructure runs today; the services are designed but not yet built.

<SystemTopology />

What this diagram is telling you, read carefully:

- **The four infra nodes are solid — they run today.** `docker compose up` brings Kafka (KRaft), Apicurio, Postgres, and Redis to a healthy state. That was the whole of Phase 0.
- **The three service nodes are dashed — they are designed, not built.** `services/{identity,chat,analytics}/` are still empty directories. Their language is fixed (TypeScript, Go, Kotlin) and their events are named in `CONTEXT.md`, but no code produces or consumes yet.
- **Every edge is dashed.** This is the honest part most architecture diagrams hide: Phase 0 stood up four _isolated_ containers. Nothing flows between them until a service exists to do the producing and consuming. Each future epic flips a node — and its edges — from planned to live.

## The contracts the edges imply

The dashed edges aren't decoration — each one is a decision already made:

- `identity → Postgres` and `identity → Kafka` together are the **transactional outbox**: Postgres holds the truth, and events flow out as a notification stream rather than the source of record.
- `Apicurio → services` is **Avro schema resolution**: the registry is the referee that keeps all three languages agreeing on the shape of every event.
- `chat → Redis` is a **read projection** off the chat log — a fast cache for the UI, never the source of truth.
